Backgroundinfos

Das Netzwerk @PhenixGames.de ist ein Projekt von 3 Jungs, darunter auch ich. Ganz am Anfang des Werdegangs, war unser Ziel einen Minecraft Server zu starten. Daraus wurde aber sehr schnell nichts. Wenig Interesse mehr an dem Server und dem Spiel, weckte uns das Interesse an einem GTA V Roleplay Server. Nach ein paar Tests, waren wir uns dann sicher, dass wir das machen möchten. Einer lernte eine komplett neue Sprache(C#) für ihn, der andere fand sich im Bereich JAVA Development wieder und ich war für die Web Präsenz zuständig. Seit dem arbeite ich sehr viel im Bereich Web Development. In der Zeit, habe ich die Offizielle Website, einen vollfunktionsfähigen Teambereich mit wichtigen, für den Game Server systemrelevanten Features und einen Userbereich für individuelle Informationen entwickelt.
Im Moment sind wir noch im Aufbau. Animation: BlackDayz
